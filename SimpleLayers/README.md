## Overview
SIMPLE LAYERS is a set of tools for Adobe After Effects designed to speed up
adding these layers to your composition:
- Null layers  
- Adjustment layers  
- Solid layers  
- 1-frame adjustment layers  

## Features
### Null Layers
- Creates a null above the selected layer  
- Trims null to match selected layer  
- Automatically parents selected layers to the new null  

### Adjustment Layers
- Creates full-length adjustment layers  
- Creates one-frame adjustment layers  

### Solid Layers
- Quickly add solid layers
- Creates one-frame solid layers
- Choose colors through a color picker  

---

## Installation
1. Download the `.jsx` script file.  
2. Place it in your After Effects Scripts folder:

**macOS:**  
`/Applications/Adobe After Effects [version]/Scripts/ScriptUI Panels/`

**Windows:**  
`C:\Program Files\Adobe\Adobe After Effects [version]\Support Files\Scripts\ScriptUI Panels\`

3. Restart After Effects.
